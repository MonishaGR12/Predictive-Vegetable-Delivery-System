# Predictive Vegetable Delivery System Using Data Science

An Android-based vegetable delivery application that combines mobile development, backend integration, and data science to improve vegetable pricing, demand prediction, and inventory planning.

This project was developed to make vegetable delivery smarter and more efficient by using machine learning to predict vegetable prices and demand trends. The system helps reduce wastage, improve stock planning, and support better business decisions for both delivery operations and admin management.



## Project Overview

The **Predictive Vegetable Delivery System Using Data Science** is a mobile application designed for online vegetable ordering and delivery management. It allows users to browse products, add items to the cart, place orders, and complete the order flow through an Android application.

In addition to standard e-commerce features, the project includes a **machine learning-based prediction module** that estimates vegetable prices and demand using historical and business-related factors. This helps improve inventory planning, pricing strategy, and operational efficiency.

The project demonstrates practical skills in:

- Android application development
- Backend development with PHP and MySQL
- Database connectivity using XAMPP
- Data science and machine learning integration
- Admin-side product and order management
- Real-world problem solving using predictive analytics
